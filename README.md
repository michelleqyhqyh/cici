# Bamboo Tales - 儿童竹纤面料服装公司网站

## 项目简介

Bamboo Tales 是一个专注于儿童竹纤面料服装的公司网站，包含以下功能：

- **公司介绍** - 展示公司logo、竹纤面料介绍和产品特色
- **设计您的衣服** - 上传设计图片，实时查看3D上身效果
- **联系我们** - 公司联系方式和地图定位

## 技术栈

- React 18 + Vite
- React Router DOM
- React Three Fiber + Drei (3D展示)
- Framer Motion (动画效果)
- Three.js (3D渲染)

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 项目结构

```
src/
├── pages/
│   ├── About.jsx          # 公司介绍页面
│   ├── Design.jsx         # 衣服设计上传和3D展示页面
│   ├── Contact.jsx        # 联系我们页面
├── App.jsx                # 主应用组件
├── App.css                # 全局样式
├── main.jsx               # 应用入口
└── index.css              # 基础样式
```
